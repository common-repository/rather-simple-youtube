A rather simple YouTube plug-in that allows the administrator to fill in a YouTube user and select a video to embed into a page.

== Description ==

This plug-in is different than other because it requires no pasting of special URL's or remembering embed codes. The first step is to fill in your own (or someone else's) YouTube username in the settings page. After this you can select any of your uploaded videos in the included widget.

== Installation ==

1. Extract the archive in wp-content/plugins
2. Activate the plugin in your WordPress administrative interface.
3. Change the settings of the plug-in
4. Insert a new video widget
